# Cold Email API

> Custom Express.js API server for the AI Cold Email Generator.

🔗 Frontend → https://cold-email-generator-livid.vercel.app

## Stack
- Node.js + Express
- Groq API (LLaMA 3.3 70B)
- Server-Sent Events for streaming

## Setup
```bash
npm install
# Add GROQ_API_KEY to .env
node server.js
```
